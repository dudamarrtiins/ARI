# Atividade Prática 1️⃣
Criação de arquitetura Cliente-Servidor 

### 💻 Métodologia Usada
- Cisco Packet Tracer


## 🌐Objetivo

  O objetivo desta tarefa foi simular uma rede corporativa funcional para a Connect-IT. Para isso, configurei um servidor centralizado responsável pela distribuição dinâmica de IPs via DHCP e pela hospedagem de uma página web customizada via HTTP. A estrutura física foi organizada em torno de um Switch central, interligando computadores via cabeamento estruturado e dispositivos móveis através de um Access Point configurado com o SSID ConnectIT_VIP. A atividade foi concluída com testes de navegação que garantiram que todos os 8 dispositivos da rede acessassem o portal interno com sucesso.
